Technology

如何实现

不依赖服务器，不收集隐私，所有计算都在你的浏览器中完成。

MediaPipe 手势识别

基于 Google MediaPipe 的纯浏览器端手势识别技术，无需安装任何软件，打开网页即可使用。所有图像处理在本地完成，不上传任何数据。

Web Speech API 语音合成

利用浏览器原生语音合成能力，将文字转化为自然流畅的语音。支持多种中文语音，为视障用户提供实时语音反馈。

PWA 离线可用

作为渐进式 Web 应用，首次访问后即可离线使用。即使在没有网络的环境下，辅助功能依然正常工作，不中断、不掉线。

零服务器成本

纯静态页面，无需后端服务器。可以部署在任何免费的静态托管平台上，零运维成本，让公益项目可持续运行。
